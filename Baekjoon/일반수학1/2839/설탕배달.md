### 설탕 배달

## 문제

```
상근이는 요즘 설탕공장에서 설탕을 배달하고 있다. 상근이는 지금 사탕가게에 설탕을 정확하게 N킬로그램을 배달해야 한다. 설탕공장에서 만드는 설탕은 봉지에 담겨져 있다. 봉지는 3킬로그램 봉지와 5킬로그램 봉지가 있다.

상근이는 귀찮기 때문에, 최대한 적은 봉지를 들고 가려고 한다. 예를 들어, 18킬로그램 설탕을 배달해야 할 때, 3킬로그램 봉지 6개를 가져가도 되지만, 5킬로그램 3개와 3킬로그램 1개를 배달하면, 더 적은 개수의 봉지를 배달할 수 있다.

상근이가 설탕을 정확하게 N킬로그램 배달해야 할 때, 봉지 몇 개를 가져가면 되는지 그 수를 구하는 프로그램을 작성하시오.
```

## 입력

```
첫째 줄에 N이 주어진다. (3 ≤ N ≤ 5000)
```

## 출력

```
상근이가 배달하는 봉지의 최소 개수를 출력한다. 만약, 정확하게 N킬로그램을 만들 수 없다면 -1을 출력한다.
```

## 예제 입력

```
4
```

## 예제 출력

```
-1
```

## 풀이

```js
가능한 적게 봉지를 사용하여야함.

그럼 상식적으로 3, 5kg의 봉지가 있을때 용량이 큰 봉지를 사용하면 봉지의 개수를 줄일 수 있다.

이는 5kg 봉지를 최대로 사용하고, 3kg봉지를 최소로 사용하여야함.

N이 주어졌을때 N을 5로 나누었을때 나머지가 0이면 남는 N없이 봉지를 사용한것.

N이 6일때 5로 나눈 나머지값은 0이 아니다. 고로 첫번째 조건문에 들어가지않음.

그럼 두번째 조건문으로 들어갔을때 6 < 0 역시 false이므로 조건문을 타지않음.

5kg봉지 사용을 못하니 3kg봉지를 사용해야함. -3 처리를 해주고 three라는 변수를 1씩 더해줌. (초기값은 0)

그리고 반복문은 계속돌아감. 6에서 3kg봉지를 하나사용했으니 3이남음. 그럼 한번더 반복문을 타고 3kg봉지를 한번더 사용함.

0이 나옴. 다시 반복문을 실행.

0 % 5는 나머지가 0임 5kg봉지를 사용하진않았지만 첫번째 조건문을 만족함.

나머지가 0이라는건 남아있는 N이 없다는것을 뜻하고 문제에서 원하는 조건을 만족시킨 봉지의 개수를 출력한다.

두번째 조건문인 input < 0은 3kg, 5kg봉지로도 해결이 안된 케이스를 뜻하고 이는 -1을 출력한다.

N이 2일때 반복문을 타면 처음에 -3처리가 되고 -1이됨 이는 두번째 조건문을타 -1을 출력하고 반복문을 종료시킨다.
```

## 내 제출

```js
const fs = require('fs');
let input = +fs.readFileSync('예제.txt').toString().trim();

let five,
    three = 0;
while (1) {
    if (input % 5 === 0) {
        five = input / 5;
        console.log(five + three);
        break;
    }

    if (input < 0) {
        console.log(-1);
        break;
    }

    input -= 3;

    three++;
}
```
