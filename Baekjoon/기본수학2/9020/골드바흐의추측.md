### 소수 구하기

## 문제

```
M이상 N이하의 소수를 모두 출력하는 프로그램을 작성하시오.
```

## 입력

```
첫째 줄에 자연수 M과 N이 빈 칸을 사이에 두고 주어진다. (1 ≤ M ≤ N ≤ 1,000,000) M이상 N이하의 소수가 하나 이상 있는 입력만 주어진다.

```

## 출력

```
한 줄에 하나씩, 증가하는 순서대로 소수를 출력한다.
```

## 예제 입력

```
3 16
```

## 예제 출력

```
3
5
7
11
13
```

## 내 제출(답은 맞지만, 시간 초과)

```
범위사이 소수를 구할 때, 소수전체가 아닌 소수의 제곱근까지만 구하면됨.

해당 답안은 시간초과가 났음.

에라토스테네스의 체로 풀어야된다고함.

const ary = [];
for (let i = +input[0]; i <= +input[1]; i++) {
    ary.push(i);
}

for (let i = 2; i <= Math.sqrt(ary[ary.length - 1]); i++) {
    for (let j = 2; i * j <= ary[ary.length - 1]; j++) {
        if (ary.indexOf(j * i) !== -1) {
            ary.splice(ary.indexOf(j * i), 1);
        }
    }
}

ary.map((v) => console.log(v));
```

## 내 제출(에라토스테네스의 체)

```
해당 문제를푼 블로그들의 답과 전혀다르게 풀었음.

아마 그렇게 푸는게 더 효율이 좋은것같음.

나는 내가 이해되는선에서 풀었음.

1. 마지막 범위를 루트처리하고 정수화 시킨다.
2. 배열을 만들어서 true로 채운다. 배열의 길이는 + 1을 해야하는데, index를 num으로 사용하기 위해서이다. 배열은 0부터 시작하니 + 1처리 했다.
3. 범위내에서 2부터 시작하여 각 수의 배수를 지움.
4. 출력할 때 입력받은 최소 범위를 포함하여 큰 index를 출력하면 답안이됨. (0, 1은 항상 제외처리)

const ary = Array(+input[1] + 1).fill(true);

for (let i = 2; i <= parseInt(Math.sqrt(+input[1])); i++) {
    for (let j = 2; i * j <= +input[1]; j++) {
        ary[i * j] = false;
    }
}

ary.map((v, i) => {
    if (i >= +input[0] && v === true && i !== 1 && i !== 0) console.log(i);
});

```
