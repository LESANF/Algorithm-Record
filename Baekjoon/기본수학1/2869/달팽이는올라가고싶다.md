### 달팽이는 올라가고 싶다

## 문제

```
땅 위에 달팽이가 있다. 이 달팽이는 높이가 V미터인 나무 막대를 올라갈 것이다.

달팽이는 낮에 A미터 올라갈 수 있다. 하지만, 밤에 잠을 자는 동안 B미터 미끄러진다. 또, 정상에 올라간 후에는 미끄러지지 않는다.

달팽이가 나무 막대를 모두 올라가려면, 며칠이 걸리는지 구하는 프로그램을 작성하시오.
```

## 입력

```
첫째 줄에 세 정수 A, B, V가 공백으로 구분되어서 주어진다. (1 ≤ B < A ≤ V ≤ 1,000,000,000)
```

## 출력

```
첫째 줄에 달팽이가 나무 막대를 모두 올라가는데 며칠이 걸리는지 출력한다.
```

## 예제 입력

```
100 99 1000000000
```

## 예제 출력

```
999999901
```

## 풀이

```js
10이라는 길이를 하루에 2씩 간다고 했을때 어떻게 구할까 ? 10/2 이다.

그냥 10에다가 2를 나눈 몫을 구하면된다.

우리는 하루에 A씩 올라가고, B씩 떨어진다. 그럼 하루에 움직이는건? A-B이다. ex) 3씩올라가고 2씩미끄러지면 하루에 1씩움직임.

길이/A-B 이다.

길이는 ? V이지만, 우리는 하루에 올라가고, 내려가고 2번 움직인다.

결국 정상에 도착했을때는 내려가는 움직임이 없다. 그렇기 때문에 길이에서 B를 빼줘야한다.
```

## 내 제출

```js
const fs = require('fs');

```

## 내 제출 오답

```js
정답은 맞지만, 문제에선 시간초과를 걸어놨기 때문에 오답처리.

const fs = require('fs');
const [a, b, v] = fs.readFileSync('/dev/stdin').toString().trim().split(' ').map(Number);

let cnt = 1;
let sum = 0;

while (sum < v) {
    sum += a;
    if (sum < v) {
        cnt++;
        sum -= b;
    } else {
        break;
    }
}

console.log(cnt);
```
